## How to Use

1. Open the webpage in your **browser** (mobile or desktop).  
2. Swipe **right** to like a cat, **left** to dislike it.  
   - On desktop, press **‚Üí** for like and **‚Üê** for dislike.  
3. After swiping all cats, a **summary** will appear in the same area showing all cats you liked.  
4. Click the **Restart** button to start swiping again.

## How It Works

- Uses **JavaScript** to load cat images and handle swipes.  
- **Touch events** handle mobile swiping.  
- **Keyboard events** handle desktop arrow key swipes.  
- Summary dynamically shows liked cats **without leaving the page**.
